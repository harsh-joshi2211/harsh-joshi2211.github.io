Project Overview

This computer vision system specializes in detecting and analyzing eyeglasses in real-time video streams. Using MediaPipe Face Mesh as a foundation, the system identifies glasses presence, estimates frame style, and tracks positioning relative to facial landmarks with 93% accuracy at 25 FPS on standard hardware.

Technical Architecture

Core Components

  • Landmark Analysis: 468-point face mesh processing
  • Frame Detection: Custom CNN for glass frame classification
  • Alignment Scoring: Geometric deviation metrics
  • Reflection Analysis: Light reflection pattern detection

Advanced Features

  • Pose-invariant detection
  • Low-light adaptation
  • Frame style recognition
  • Wear-time monitoring

Detection Methodology

The system employs a multi-stage verification process:

  1. Nose Bridge Analysis: Measures pressure points and frame curvature
  2. Temple Detection: Tracks arms extending toward ears
  3. Reflection Signature: Identifies characteristic glass reflections
  4. Occlusion Patterns: Analyzes eyebrow and eyelid obscuration

Innovative Algorithms

def detect_eyeglasses(face_landmarks):
    # Nose bridge analysis
    nose_points = [face_landmarks[i] for i in [168, 6, 197]]
    bridge_curve = calculate_curvature(nose_points)
    
    # Temple detection
    left_temple = analyze_region(face_landmarks, LEFT_TEMPLE_INDICES)
    right_temple = analyze_region(face_landmarks, RIGHT_TEMPLE_INDICES)
    
    # Reflection analysis
    reflection_score = assess_reflections(
        face_landmarks[FACE_OVAL_INDICES], 
        frame_gray
    )
    
    # Combined decision
    return (0.4*bridge_curve + 
            0.3*(left_temple + right_temple)/2 + 
            0.3*reflection_score) > GLASSES_THRESHOLD

Challenges Overcome

Key technical hurdles addressed:

  • Differentiating actual glasses from similar facial features
  • Handling various frame materials (plastic, metal, rimless)
  • Accounting for reflections and glare
  • Maintaining performance with accessories (hats, scarves)
